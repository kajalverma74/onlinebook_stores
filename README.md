# Online Book Store

This is an online book store application built with **Node.js**, **Express.js**, and **MongoDB**. It allows users to browse, search, and purchase books.

## Features
- User registration and login with authentication (JWT).
- Book catalog with images (`book thief.jpeg`, `godan.jpeg`).
- Shopping cart functionality.
- Secure payments (to be integrated).

## Project Structure
- `server.js`: Main application file.
- `models/`: Contains MongoDB models (e.g., User, Book).
- `routes/`: API endpoints for user and book-related operations.
- `uploads/`: Directory for storing book images.
- `.gitignore`: Excludes `node_modules/` and `.env` file.

## Getting Started

### Prerequisites
- Node.js
- MongoDB
- npm
### Install dependencies:
 npm install
### Start the server:
   npm start
